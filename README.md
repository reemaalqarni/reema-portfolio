# Energy Analysis Project ⚡

This project analyzes the relationship between mass, velocity, and energy using the formula:

E = 0.5 × m × v²

Tools used:
- Excel for data preparation
- Power BI for visualization

The dashboard includes:
- Line chart showing Energy vs Time
- Trend line
- Average line
- Physics explanation text box
